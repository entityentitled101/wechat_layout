# 微信公众号自动排版配图工具

这是一个专门为微信公众号设计的工具包，旨在解决 AI 生成的 Markdown 内容在微信后台排版乱、格式不统一的问题。目前提供强大的 **Chrome 浏览器扩展程序**，可直接在微信公众号编辑器中实现一键排版。

## 🌟 核心功能特性

### 1. 微信编辑器内直连排版 (Chrome 扩展)
- **物理隔离渲染引擎**：插件内置独立的安全 Markdown 解析器，完美避开微信编辑器自带复杂脚本的加载冲突（不再依赖外部环境）。
- **空行压实与间距重构**：彻底解决渲染后段落间距过大的问题。算法可智能过滤冗余换行，死死锁定最舒适的段落纵向呼吸感，使排版紧密、流畅而不拥挤。
- **悬浮注入一键直达**：在微信编辑器右侧生成“魔法自动排版”悬浮按钮。全自动抓取内容、格式化、带样式回填，无需切换网页、无需繁琐复制。

### 2. 严苛的美学规范与精准识别
- **数字阅读友好字号**：正文 15px，无衬线字体栈优先匹配（苹方/微软雅黑），1.8 倍优享行高。
- **标题视觉体系**：一级标题 18px 居中强调；二、三级标题 17px 专属左绿色边框醒目指引。
- **强化型引言识别 (Quote)**：针对 `>` 以及全角 `＞` 做了深度正则表达式匹配，全覆盖转化为带品牌绿及圆角的精美引言小卡片。
- **语义化特殊卡片**：自动探知文章中的“互动话题”、“推荐阅读”、“参考书目”等版块，自适应包装为浅蓝虚线框或低饱和灰色附注卡片。
- **清理与修饰**：自动清理系统不支持的残余 Emoji，并将分隔符 `---` 优雅替换为带有“✦ ✦ ✦”元素的宽宏留白线。

## 🚀 Chrome 扩展安装与使用指南

该插件完全依赖本地运作，极为安全，使用步骤如下：

1. 打开 Chrome 浏览器，在地址栏输入访问 `chrome://extensions/`。
2. 确保页面右上角的 **“开发者模式”** 处于开启状态。
3. 点击左上角 **“加载已解压的扩展程序”**。
4. 在弹出的本地目录中，选中本项目的 `plugin_wechat_layout` 文件夹并确定。
5. 登录 [微信公众平台](https://mp.weixin.qq.com/) 进入图文编辑界面，或按下 `Ctrl + F5` 强制刷新当前编辑器网页。
6. 将需要排版的 Markdown 原料文本粘贴到编辑器中。
7. 点击页面右侧悬浮的 **“✨ 魔法自动排版”** 按钮，格式瞬间完成转化！
