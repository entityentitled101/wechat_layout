# 微信公众号自动排版工具 (WeChat Auto-Typesetter)

这是一个专门为微信公众号设计的排版工具，旨在解决 AI 生成的 Markdown 内容在微信后台排版乱、格式不统一的问题。

## 核心特性

- **一键清理**：自动移除所有 Emoji，保持内容干练。
- **精准字号与审美 (v1.0 稳定版)**：
  - **正文**：严格锁定 15px，微软雅黑/苹方字体栈，1.75 倍白银比例行高。
  - **标题**：一级标题 18px 居中；二、三级标题 17px 带左绿边框。
  - **引言 (Quote)**：17px 绿标提示 + 15px 黑色正文，解决微信后台渲染错乱问题。
  - **特殊板块**：自动识别「参考资料」、「互动话题」、「参考书目」，并渲染为 12px 的灰色精美卡片。
  - **分割线**：✦ ✦ ✦ 居中装饰风格。
- **微信原生适配**：全内联样式注入，复制到微信后台后效果 100% 还原。

## 当前已知问题与待解决 (Pending Issues)

- **非标准表格解析**：
  - 目前版本对于标准的 Markdown 表格支持良好。
  - **局限性**：对于 AI 生成的、被空行切断或文字换行导致的“支离破碎”表格，目前的“碎片重组”算法由于逻辑过于激进可能导致渲染失败。
  - **现状**：已暂时移除复杂的表格修复逻辑，优先保证全文内容的显示稳定性。如果遇到表格错位，建议手动调整 Markdown 源内容使其符合标准规范。

## 本地运行指南

1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```

## 使用方法
1. 将 Markdown 内容粘贴到左侧输入框。
2. 点击“一键排版”。
3. 在右侧预览效果并点击“复制 HTML”粘贴到微信后台。
