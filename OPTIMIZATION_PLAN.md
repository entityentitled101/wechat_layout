# 微信公众号排版插件优化计划

## 需求分析总结

### 1. 标题字号体系重构
- **现状问题**：一级标题 18px，二级标题 17px，差别太小；三级标题也是 17px
- **目标**：
  - 一级标题：比二级标题大 2 个字号（默认）
  - 二级标题：保持现有 17px
  - 三级标题：和正文同样字号（15px）
  - 支持用户自定义调整

### 2. 三明治菜单（设置面板）
- 汉堡菜单按钮 ☰ 打开侧边设置面板
- 面板内包含：
  - 一级/二级/三级标题字号调节（滑块或数字输入）
  - 三横线样式选择
  - 加粗文字颜色选择
  - 特殊框样式选择
  - 颜色主题选择

### 3. 自动排版按钮重构
- 从大的魔法按钮变成紧凑型工具栏
- 布局：「魔法自动排版 | ▶ | ☰」
  - 文字部分："魔法自动排版"
  - 播放按钮：▶ 直接执行排版
  - 汉堡菜单：☰ 打开设置面板

### 4. 三横线逻辑优化
- **规则**：
  - 三级标题前：不加三横线
  - 一级/二级标题前：如果前面有正文文本，自动加三横线；如果前面没有其他内容或已经是分隔符，不加
- **样式选项**：提供 3-5 种分割线样式供选择

### 5. Markdown 格式全覆盖
需要支持的 Markdown 元素清单：
- [x] 一级标题 (#)
- [x] 二级标题 (##)
- [x] 三级标题 (###)
- [x] 引用 (>)
- [x] 无序列表 (-, *)
- [ ] 有序列表 (1. 2. 3.)
- [ ] 加粗 (**text**) - 需要支持颜色变化
- [ ] 斜体 (*text*)
- [x] 分割线 (---, ***)
- [x] 表格 (| | |)
- [ ] 行内代码 (`code`)
- [ ] 代码块 (```code```)
- [ ] 删除线 (~~text~~)
- [ ] 链接 ([text](url))
- [ ] 图片 (![alt](url))

### 6. 特殊文字样式
- **加粗变绿色**：可选开关，开启后 **加粗** 文字显示为品牌绿 #07c160
- **书名号《》样式**：可选高亮或变色
- **括号样式**：圆括号、方括号特殊样式
- **选中文字快捷样式**：在面板中增加"选中文字样式"区域，可以：
  - 转为引用样式
  - 转为高亮框
  - 转为提示框
  - 转为警告框

### 7. 多种框样式设计
设计 6-8 种精美框样式（使用微信品牌绿色系）：
1. 引言框（已有）- 左侧绿条
2. 提示框 - 浅绿背景
3. 信息框 - 浅蓝背景
4. 警告框 - 浅黄背景
5. 重点框 - 浅灰背景+边框
6. 引用框2 - 双边框
7. 卡片框 - 阴影卡片
8. 简约框 - 只有底部边框

### 8. 颜色体系（微信品牌色）
```
主色：#07c160 (微信绿)
深色：#06ae56
浅色：#e6f7ed
背景灰：#f5f7fa
文字深：#1f2937
文字中：#4b5563
文字浅：#9ca3af
分割线：#e1e8ed
```

## 实施步骤

### Phase 1: 核心架构重构
1. 创建设置管理器 (SettingsManager) - 管理所有用户配置
2. 创建样式生成器 (StyleGenerator) - 根据配置生成 CSS
3. 重构标题渲染逻辑 - 支持动态字号

### Phase 2: UI 界面改造
1. 重新设计工具栏按钮（魔法自动排版 + 播放 + 菜单）
2. 实现三明治菜单侧边面板
3. 实现各种设置控件（滑块、选择器等）

### Phase 3: 排版逻辑增强
1. 实现三横线智能插入逻辑
2. 实现所有 Markdown 格式支持
3. 实现加粗变色、书名号样式等

### Phase 4: 框样式系统
1. 设计并实现 6-8 种框样式
2. 实现选中文字应用框样式功能
3. 在菜单中提供框样式预览和选择

### Phase 5: 同步到 Chrome 扩展
1. 将新功能同步到 `content_scripts/wechat_typeset.js`
2. 更新 popup 窗口界面
3. 测试扩展版功能

## 技术实现要点

1. **设置持久化**：使用 localStorage 保存用户设置
2. **实时预览**：修改设置后立即更新预览
3. **选中文字处理**：使用 Selection API 获取选中内容
4. **样式隔离**：所有样式使用内联 style 确保微信编辑器兼容
