# 微信公众号排版助手 V2.6 更新日志

## 🎉 V2.6 精细排版更新

### 1. 行间距可调
**问题**：一级标题下面的空行间距太大

**解决方案**：
添加三种行间距模式：
- **紧凑** (compact)：段落间距 12px，标题间距 16-22px
- **标准** (normal)：段落间距 16px，标题间距适中
- **宽松** (loose)：段落间距 25px（原版）

**使用方式**：
打开设置面板 (☰) → 选择「行间距」→ 紧凑/标准/宽松

### 2. 单独加粗功能
**需求**：选中一段文字，单独给它加粗（绿色）

**使用方式**：
- **快捷键**：选中文字后按 `Alt+0`
- **按钮**：打开设置面板 → 点击「𝐁 加粗文字」按钮

**效果**：`<strong style="color: #07c160; font-weight: bold;">选中的文字</strong>`

### 3. 单独插入分割线
**需求**：在任意位置插入分割线

**使用方式**：
打开设置面板 → 点击「✦✦✦ 插入分割线」按钮

**分割线样式**：跟随设置中的「三横线样式」选择

### 4. 图片解释文字
**需求**：图片说明文字，居中、绿色、比正文小2个字号

**新框样式**：
- **名称**：图片解释
- **快捷键**：`Alt+9`
- **图标**：🖼️
- **样式**：
  - 居中显示 `text-align: center`
  - 绿色文字 `color: #07c160`
  - 字号 13px（正文15px小2个）
  - 斜体 `font-style: italic`

**使用方式**：
1. 选中图片下方的说明文字
2. 按 `Alt+9` 或点击「🖼️ 图片解释」按钮

## 📖 使用示例

### 示例1：紧凑排版
```
设置面板 → 行间距 → 紧凑
→ 段落间距变小，更紧凑
```

### 示例2：图片说明
```
1. 插入图片
2. 在图片下方输入说明文字
3. 选中说明文字
4. 按 Alt+9
→ 文字变成居中绿色小字
```

### 示例3：单独加粗
```
1. 选中一段重要文字
2. 按 Alt+0
→ 文字变成绿色加粗
```

### 示例4：插入分割线
```
1. 光标放在想要分割的位置
2. 打开设置面板
3. 点击「✦✦✦ 插入分割线」
→ 插入当前设置的分割线样式
```

## 🐛 修复内容

1. 行间距太大 → 添加紧凑模式
2. 无法单独加粗 → 添加 Alt+0 快捷键
3. 无法单独插分割线 → 添加分割线按钮
4. 缺少图片说明样式 → 添加图片解释框

## 🔧 技术细节

### 行间距设置
```javascript
paragraphSpacing: 'compact' | 'normal' | 'loose'

// 紧凑模式
margin-bottom: 12px;  // 段落
margin: 16px 0;       // 框样式
margin: 22px 0 12px;  // 一级标题

// 标准模式
margin-bottom: 16px;
margin: 16px 0;
margin: 22px 0 12px;

// 宽松模式
margin-bottom: 25px;
margin: 25px 0;
margin: 35px 0 20px;
```

### 图片解释样式
```javascript
{
  name: '图片解释',
  render: (content, color, icon) => `
    <div style="text-align: center; margin: 12px 0;">
      <span style="font-size: 13px; color: ${color}; font-style: italic; line-height: 1.6;">
        ${icon} ${content}
      </span>
    </div>
  `
}
```
