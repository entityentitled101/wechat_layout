# 微信公众号排版助手 V2.5 更新日志

## 🐛 修复的问题

### 1. 恢复 Alt+数字快捷键
**问题**：豆包浏览器中点击应用框样式失效

**解决方案**：
- 恢复 Alt+1~8 快捷键应用框样式
- 在框样式按钮上显示对应的快捷键提示

| 快捷键 | 框样式 | 图标 |
|--------|--------|------|
| Alt+1 | 引言框 | 💬 |
| Alt+2 | 提示框 | 💡 |
| Alt+3 | 信息框 | ℹ️ |
| Alt+4 | 警告框 | ⚠️ |
| Alt+5 | 重点框 | 🎯 |
| Alt+6 | 优雅框 | ✨ |
| Alt+7 | 简约框 | ▪️ |
| Alt+8 | 卡片框 | 📋 |

### 2. 修复选中文字应用框样式
**问题**：点击框样式按钮没有反应

**解决方案**：实现多种备选插入方式：
1. 尝试在当前文档执行 `execCommand`
2. 尝试在 iframe 中执行 `execCommand`
3. 直接替换选中的文本节点

### 3. 修复设置面板遮挡内容
**问题**：面板固定无法滚动

**解决方案**：
- 面板最大高度限制为 `70vh`
- 面板位置固定在 `top: 270px`
- 面板内容可滚动
- 关闭面板时自动退出编辑模式

## 🎉 功能特性

### 1. 动态标题字号
- 一级标题：默认 20px（16-28px 可调）
- 二级标题：默认 17px（14-22px 可调）
- 三级标题：默认 15px（13-19px 可调，与正文相同）

### 2. 三横线样式
- ✦✦✦ 星星（默认）
- ─── 横线
- ··· 圆点

### 3. 文字样式开关
- 加粗文字变绿色
- 书名号《》高亮

### 4. 框样式系统（8种）
支持 Alt+1~8 快捷键快速应用

### 5. 框样式图标自定义
**编辑图标模式**：
1. 打开设置面板
2. 点击「✏️ 编辑图标」按钮
3. 点击任意框样式更换图标
4. 点击「✓ 完成」退出

## 📖 使用方式

### Markdown 排版
1. 粘贴 Markdown 到公众号编辑器
2. 点击「✨ 魔法自动排版」或 ▶ 按钮

### 应用框样式（两种方式）

**方式1：快捷键（推荐）**
1. 在编辑器中选中文字
2. 按 Alt+1~8 快速应用框样式

**方式2：点击按钮**
1. 在编辑器中选中文字
2. 点击 ☰ 打开设置面板
3. 点击框样式按钮应用

### 更换框样式图标
1. 点击 ☰ 打开设置面板
2. 点击「✏️ 编辑图标」
3. 点击框样式 → 选择新图标
4. 点击「✓ 完成」退出
5. 保存设置
